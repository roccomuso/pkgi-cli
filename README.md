> Node.js interactive pkg CLI tool

To embed it in your Node.js app use the [pkgi](https://github.com/roccomuso/pkgi) package instead. This is meant to be used as a standalone tool.

## Install

    $ npm install -g pkgi-cli

## Usage

    $ pkgi
